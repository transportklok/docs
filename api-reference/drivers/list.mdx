Met dit endpoint haalt u een gepagineerde lijst op van alle chauffeurs die zijn geregistreerd in uw Transportklok account. U kunt de resultaten verfijnen met optionele filters op beschikbaarheidsstatus of rijbewijsklasse, zodat u snel de juiste chauffeur vindt voor een geplande rit.

## Endpoint

```
GET https://api.transportklok.nl/v1/drivers
```

## Authenticatie

Voeg de volgende header toe aan elk verzoek:

```
Authorization: Bearer <api_key>
```

## Queryparameters

<ParamField query="status" type="string">
  Filter chauffeurs op beschikbaarheidsstatus. Toegestane waarden: `available`, `on_trip`, `unavailable`, `inactive`.
</ParamField>

<ParamField query="license_class" type="string">
  Filter chauffeurs op rijbewijsklasse. Voorbeelden: `B`, `C`, `CE`. Geeft alle chauffeurs terug die ten minste de opgegeven klasse bezitten.
</ParamField>

<ParamField query="page" type="integer" default="1">
  Het paginanummer om op te halen. Begint bij `1`.
</ParamField>

<ParamField query="per_page" type="integer" default="25">
  Het aantal resultaten per pagina. Maximum: `100`.
</ParamField>

## Voorbeeldverzoek

```bash
curl --request GET \
  --url "https://api.transportklok.nl/v1/drivers?status=available&license_class=CE&page=1&per_page=25" \
  --header "Authorization: Bearer <api_key>" \
  --header "Content-Type: application/json"
```

## Responsvelden

<ResponseField name="data" type="array">
  Een array van chauffeurobjecten die voldoen aan de opgegeven filtercriteria.

  <Expandable title="Chauffeurobject">
    <ResponseField name="id" type="string">
      De unieke identifier van de chauffeur, voorafgegaan door `drv_`.
    </ResponseField>

    <ResponseField name="name" type="string">
      De volledige naam van de chauffeur.
    </ResponseField>

    <ResponseField name="email" type="string">
      Het e-mailadres van de chauffeur.
    </ResponseField>

    <ResponseField name="phone" type="string">
      Het telefoonnummer van de chauffeur in internationaal formaat (E.164).
    </ResponseField>

    <ResponseField name="license_classes" type="array">
      Een array van rijbewijsklassen die de chauffeur bezit. Mogelijke waarden: `B`, `C`, `CE`.
    </ResponseField>

    <ResponseField name="status" type="string">
      De huidige beschikbaarheidsstatus van de chauffeur. Mogelijke waarden: `available`, `on_trip`, `unavailable`, `inactive`.
    </ResponseField>

    <ResponseField name="created_at" type="string">
      Tijdstip waarop het chauffeursprofiel is aangemaakt, in ISO 8601-formaat.
    </ResponseField>

    <ResponseField name="updated_at" type="string">
      Tijdstip van de laatste wijziging aan het chauffeursprofiel, in ISO 8601-formaat.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="meta" type="object">
  Metadata over de paginering van de resultatenset.

  <Expandable title="Meta-object">
    <ResponseField name="meta.current_page" type="integer">
      Het huidige paginanummer van de geretourneerde resultaten.
    </ResponseField>

    <ResponseField name="meta.per_page" type="integer">
      Het aantal resultaten per pagina zoals opgegeven in het verzoek.
    </ResponseField>

    <ResponseField name="meta.total" type="integer">
      Het totale aantal chauffeurs dat voldoet aan de filtercriteria.
    </ResponseField>
  </Expandable>
</ResponseField>
