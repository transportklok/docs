## Doel van de pagina

Deze pagina documenteert de tijdregistratieflow voor medewerkers.

## Wat je hier kunt doen

- Het overzicht openen
- Een registratie aanmaken
- Details bekijken
- De tijdlijn bekijken
- Een registratie bewerken

## Belangrijkste onderdelen

- **Overzicht** via route `workforce.hour-management.time-registrations`
- **Aanmaken** via route `workforce.hour-management.time-registrations.create`
- **Details** via route `workforce.hour-management.time-registrations.show`
- **Tijdlijn** via route `workforce.hour-management.time-registrations.timeline`
- **Bewerken** via route `workforce.hour-management.time-registrations.edit`
